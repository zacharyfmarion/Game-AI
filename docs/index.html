Welcome to Game AI’s documentation!¶

GameAI contains a series of well-defined abstractions that are common in AI, such as a games, agents, and trainers which optimize the behavior of agents. As long as a class inherits from the base implemetation of a primitive (e.g. Agent) and implements the required methods, it can be used in place of the standard implementations given.
