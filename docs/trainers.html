Trainers¶

Trainers train an agent on a game

class trainers.Trainer[source]¶

Class that provides an interface for training an agent. This is necessary because agents are bound to a particular player, but for some algorithms the agent is really being trained to play optimally for both plays, so we have this class house the training data and then pass it into the agents when they are instantiated to avoid duplicated work

train(g, **kwargs)[source]¶

Train the agent. As a convenience this should return self.training_params() at the end of training
